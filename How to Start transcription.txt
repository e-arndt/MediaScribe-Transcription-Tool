MediaScribe - Quick Start

MediaScribe creates transcripts and caption files from audio/video files.

There are two main steps:

1. Install MediaScribe
2. Use MediaScribe

============================================================

1. INSTALL MEDIASCRIBE
   ============================================================

Open the folder where you downloaded or copied MediaScribe.

This might be a USB drive, Downloads folder, Desktop folder, project folder, or extracted ZIP folder.

In that folder, double-click:

Install.bat

or:

Install-MediaScribe.bat

The installer will start PowerShell and run setup.ps1 for you.

During setup, choose where to install MediaScribe.

The default install folder is usually:

Documents\MediaScribe

Setup will create the installed MediaScribe folder and these subfolders:

Input
Output
Logs
Models
Tools

Setup will also create:

MediaScribe.bat

That is the normal launcher you use after installation.

If setup completes successfully and all required dependencies are available, the installer may ask:

Run MediaScribe now? (Y/N, default Y)

Press Enter or choose Y to start MediaScribe immediately.

Choose N if you want to run MediaScribe later.

============================================================
2. REINSTALLING OR REPAIRING MEDIASCRIBE
========================================

You can run Install.bat again to reinstall or repair MediaScribe.

This is useful if core app files were damaged, deleted, or need to be updated.

During reinstall or repair, setup may replace the core app files, including:

transcribe.ps1
README.md
How to Start transcription.txt
MediaScribe.bat
config.json

Setup recreates required folders if they are missing.

Setup does not clear or delete files inside these folders:

Input
Output
Logs
Models
Tools

This means existing media files, transcripts, WAV files, caption files, and logs should remain in place.

To remove MediaScribe, delete the installed MediaScribe folder.

============================================================
3. DEPENDENCIES
===============

MediaScribe requires:

Python
pip
OpenAI Whisper
FFmpeg

The setup script checks for these dependencies.

FFmpeg may be bundled with MediaScribe in:

Dependencies\FFmpeg

During setup, bundled FFmpeg is copied into the installed app folder here:

Tools\ffmpeg

MediaScribe looks for FFmpeg in this order:

1. Local Tools\ffmpeg\ffmpeg.exe
2. System FFmpeg from PATH

Python may be bundled with MediaScribe in:

Dependencies\Python

If Python is missing, setup can offer to run the bundled Python installer.

pip is handled through Python. If pip is missing, setup can try to repair it using Python's ensurepip feature.

OpenAI Whisper is installed with pip if it is missing.

Installing Whisper requires internet access.

MediaScribe looks for Whisper in this order:

1. python -m whisper
2. Global whisper.exe / whisper command

If setup says a dependency is missing, MediaScribe files may still be installed, but transcription will not work until the missing dependency is installed or fixed.

After fixing missing dependencies or internet access, run Install.bat again to re-check setup.

============================================================
4. USE MEDIASCRIBE
==================

After setup finishes, you can use MediaScribe right away if setup asks to run it.

To use MediaScribe later, open the installed MediaScribe folder.

Example:

Documents\MediaScribe

Put one or more audio or video files into the Input folder.

Recognized video files:

MP4, MKV, MOV, M4V, AVI, WEBM

Recognized audio files:

MP3, M4A, WAV, AAC, FLAC, OGG, OPUS, WMA

Then start MediaScribe by double-clicking:

MediaScribe.bat

You can also run it from PowerShell with:

.\MediaScribe.bat

Install.bat is for installing, reinstalling, or repairing MediaScribe.

MediaScribe.bat is the normal launcher for using MediaScribe after installation.

============================================================
5. START SCREEN
===============

When MediaScribe starts, it shows a welcome screen with:

What MediaScribe does
Default output information
Full output information
Recognized video formats
Recognized audio formats
Input folder location
Output folder location

Press Enter to continue.

============================================================
6. CHOOSE WHAT TO TRANSCRIBE
============================

MediaScribe will show a list of recognized files.

Example:

[1] First file.mp4
[2] Second file.mp3
[B] Process all files
[Q] Quit

Choose a file number to process one file.

Choose B to process all files.

Choose Q to quit.

If no files are found, you can add files to the Input folder and choose Refresh.

[R] Refresh / check again
[O] Open Input folder
[Q] Quit

Pressing Enter refreshes.

============================================================
7. CHOOSE OUTPUT FILES
======================

MediaScribe will ask what output files you want.

[D] Default - Original file, WAV audio, TXT transcript
[F] Full    - Original file, WAV audio, TXT, SRT, VTT, TSV, JSON

Default mode is recommended for most users.

Use Full mode when you need caption/subtitle files or detailed Whisper output.

Caption use:

Use SRT or VTT files for video caption/subtitle workflows.

============================================================
8. CHOOSE TRANSCRIPTION MODE
============================

MediaScribe will ask which transcription mode to use.

[F] Fast     - Uses the configured/default Whisper model
[A] Accurate - Uses the large Whisper model

Pressing Enter uses Fast mode.

Fast mode is recommended for most users.

Accurate mode is slower and may help when audio is harder to understand or Fast mode misses words.

============================================================
9. WHILE TRANSCRIBING
=====================

MediaScribe first extracts audio with FFmpeg.

Then MediaScribe runs OpenAI Whisper.

During Whisper transcription, live transcript progress appears below the Transcribing section.

This progress may appear in chunks and may take a while, especially for longer files or larger Whisper models.

This is normal.

============================================================
10. FIND RESULTS
================

Results are saved in the Output folder.

Each file gets its own output folder.

Example:

Output\My Recording
My Recording.mp3
My Recording.wav
My Recording.txt

In Default mode, MediaScribe creates:

Original file
WAV audio
TXT transcript

In Full mode, MediaScribe creates:

Original file
WAV audio
TXT transcript
SRT subtitle file
VTT subtitle file
TSV timing file
JSON Whisper output

============================================================
11. AFTER PROCESSING
====================

After processing finishes, MediaScribe asks what to do next.

[R] Run again / return to file list
[O] Open output folder
[Q] Quit

Pressing Enter quits.

============================================================
NOTES
=====

DRM/protected media may fail.

Audio is extracted only and converted to mono 16 kHz WAV for consistency.

Files inside the Input folder are moved into their output folder after successful processing.

External files processed through command-line parameter mode are left in place.

Install.bat or Install-MediaScribe.bat is only used to install, reinstall, or repair MediaScribe.

After installation, use MediaScribe.bat from the installed MediaScribe folder.
